# Giới thiệu

Đây là một bài viết mẫu để demo các tính năng của blog. Bài viết này sử dụng MDX để viết nội dung.

## Tính năng 1

MDX cho phép bạn sử dụng các component React trong markdown:

```jsx
<Box sx={{ p: 2, bgcolor: 'primary.main', color: 'white' }}>
	Đây là một component React trong MDX
</Box>
```

## Tính năng 2

Bạn có thể sử dụng các tính năng markdown thông thường:

- Danh sách
- **In đậm**
- _In nghiêng_
- [Link](https://example.com)

### Code blocks

```typescript
function hello() {
	console.log('Hello, world!');
}
```
